# Polaris Rover System Design

## Mission

Design a small-scale planetary rover prototype capable of autonomous movement and environmental sensing.

## Version

Polaris Rover V1.0

## Core Systems

### Mobility System
- 4WD chassis
- DC motors
- Motor driver

### Control System
- Arduino Uno

### Sensor System
- DHT11 temperature and humidity sensor
- Soil moisture sensor
- Water detection sensor

### Future Upgrades
- Camera system
- Computer vision
- ESP32 communication
- Autonomous navigation

## Design Philosophy

The rover will be developed through multiple iterations, with each version improving reliability and capability.

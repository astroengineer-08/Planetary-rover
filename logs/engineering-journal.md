# Engineering Journal

## July 2026 — Project Start

### Goal
Create a robotics platform inspired by planetary exploration rovers.

### Current Resources

Available:
- Arduino Uno
- Soil moisture sensor
- Water level sensor
- DHT11 temperature/humidity sensor
- DC motors
- Water pump motors
- Jumper wires

### Current Challenges

- Selecting a suitable rover chassis
- Designing the motor control system
- Understanding power requirements

## Power System Investigation

Initial testing will use a 3xAAA battery holder (~4.5V).

Limitations:
- Suitable for prototype motor testing
- Insufficient for final rover due to motor current requirements

Future upgrade:
- Higher current rechargeable battery system
- Dedicated motor and logic power management

### Next Steps

- Purchase rover chassis
- Purchase motor driver
- Build first movement prototype
- Document hardware decisions
